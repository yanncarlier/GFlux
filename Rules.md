## Rules Summary (The "Must-Knows")  

The Tech: You must use the Gemini 2.0 Flash model via the Multimodal Live API (WebSockets).  

The Categories: You can focus on Multimodal Interaction (voice/video) or Task-Oriented Agents (Function Calling).  



All projects MUST: 

Leverage a Gemini model

Agents must be built using either Google GenAI SDK OR ADK (Agent Development Kit)

https://docs.cloud.google.com/vertex-ai/generative-ai/docs/learn/overview  

https://google.github.io/adk-docs/
